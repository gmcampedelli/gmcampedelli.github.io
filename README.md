# gcampede.github.io

Personal academic website of **Gian Maria Campedelli** — Assistant Professor, University of Trento; Fellow, Fondazione Bruno Kessler.

Static site (plain HTML/CSS), served via GitHub Pages at <https://gcampede.github.io>.

## Structure
- `index.html` — home (bio + recent news)
- `cv.html` — CV highlights + PDF
- `papers.html` — publications by research area, with abstracts
- `teaching.html` — courses
- `media.html` — media coverage, outreach & talks
- `assets/` — stylesheet, photo, CV

## Editing
All colors are CSS variables at the top of `assets/style.css`. Bump the `?v=` query on the stylesheet link when you change CSS to bypass browser caching.
